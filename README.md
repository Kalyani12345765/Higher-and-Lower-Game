# Higher-and-Lower-Game
A Higher or Lower game built with Python as part of the 100 Days of Code: The Complete Python Pro Bootcamp by Dr. Angela Yu.

A Simple Game where user can guess who has more Instagram followers, and based on that hitting up the scores
Inspired by *Higher and Lower Game* :)

I wrote this version of the game **independently**, using my own logic and structure.
The dataset and original project concept are provided as part of the course materials.
